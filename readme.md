# 🩺 Diabetes Prediction App

## 🚀 Overview
This project is a Machine Learning-based web application that predicts diabetes disease progression using medical attributes.

The model is deployed using Streamlit and provides an interactive interface for users to input patient data and get predictions instantly.

---

## 🧠 Features
- Interactive UI using Streamlit
- Sidebar-based user input
- Real-time prediction
- Input summary display
- Feature importance visualization

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit

---

## 📊 Model Details
- Algorithm: Decision Tree Regressor
- Criterion: Absolute Error
- Max Depth: 5
- Splitter: Random

---

## ▶️ How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
